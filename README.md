# 🍕 Fast React Pizza

A fully responsive, interactive pizza ordering web app built entirely with **React**. Users can browse a dynamic menu, add items to their cart, and place orders with live form validation and order tracking.

<br />

## 🔗 Live Demo

🚀 [**View Live App**](https://fastreactpizza-sigma.vercel.app/) – Deployed on **Vercel**

<br />

## ⚙️ Tech Stack

- ⚛️ **React**
- 🔁 **React Router** for navigation
- 📦 **State Management** with `useContext` (or Redux alternative)
- ☁️ **Vercel** for fast deployment and hosting

<br />

## ✨ Key Features

✅ Interactive pizza menu  
🛒 Dynamic cart with real-time updates  
📝 Delivery form with validation  
📦 Order summary before checkout  
⚡ Fast and responsive on all devices
